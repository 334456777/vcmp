# vcmp - 视频静态场景检测工具

vcmp 是一个用于检测视频中静态场景的命令行工具。通过逐帧分析视频内容，自动识别画面静止的片段，并生成 FCPXML 标记文件供 Final Cut Pro X 使用。

## 功能特性

- 🎬 **自动检测静态场景**：使用帧差法逐帧分析视频，识别画面静止的片段
- 📊 **智能阈值建议**：基于 P95 百分位数自动计算建议阈值
- 💾 **分析数据缓存**：首次分析后生成 `.gob` 文件，后续操作无需重新分析
- 🎯 **FCPXML 导出**：生成 Final Cut Pro X 兼容的标记文件，可直接导入使用
- ⚡ **高性能处理**：使用并发处理和对象池优化，提升分析速度
- 🎨 **字幕区域排除**：自动裁剪底部区域，避免硬编码字幕干扰检测

## 安装

### 前置要求

- Go 1.25.2 或更高版本
- OpenCV（通过 gocv.io/x/gocv 使用）

### 构建

```bash
# 克隆仓库
git clone <repository-url>
cd vcmp

# 构建二进制文件
make build

# 或使用 go 命令
go build -o vcmp .
```

### 安装到系统路径

```bash
# 安装到 /opt/homebrew/bin（macOS）
make install

# 需要 sudo 权限
```

## 使用方法

### 基本工作流程

1. **首次分析视频**：将视频文件放在当前目录，运行 `vcmp`
   ```bash
   vcmp
   ```
   程序会自动检测当前目录下的视频文件（.mp4、.mov 等），进行分析并生成 `.gob` 分析数据文件。

2. **查看分析统计**：直接运行 `vcmp` 查看已有 `.gob` 文件的分析结果
   ```bash
   vcmp
   ```

3. **生成 FCPXML 标记**：指定阈值生成 FCPXML 文件
   ```bash
   vcmp <threshold>
   ```
   或同时指定阈值和最小持续时间：
   ```bash
   vcmp <threshold> <min_duration>
   ```

### 参数说明

- `threshold`：差异像素数阈值。当连续帧的差异像素数低于此阈值时，判定为静态场景
- `min_duration`：最小持续时间（秒）。只有持续时间达到此值的静态片段才会被标记（默认：20秒）

### 使用示例

```bash
# 分析视频并生成 .gob 文件
vcmp

# 使用建议阈值生成 FCPXML（假设建议阈值为 1000）
vcmp 1000

# 使用自定义阈值和最小持续时间
vcmp 800 15

# 只标记持续时间超过 30 秒的静态片段
vcmp 1000 30
```

## 工作原理

### 检测算法

1. **帧差法**：比较相邻帧的灰度图差异，统计变化像素数量
2. **二值化处理**：使用阈值（默认 25）将差异图二值化
3. **形态学处理**：使用腐蚀操作去除噪声
4. **片段识别**：当连续多帧的差异像素数低于阈值时，判定为静态片段

### 阈值计算

程序使用 P95（第95百分位数）× 1.5 作为默认建议阈值。这个策略可以：
- 过滤掉大部分正常的画面抖动
- 保留真正的静态场景
- 根据视频内容自动调整

### 字幕区域处理

程序会自动裁剪掉画面底部 65/1080 的区域（约 6%），用于排除：
- 硬编码字幕
- 水印
- 其他固定位置的 UI 元素

## 输出文件

### .gob 文件

分析数据文件，包含：
- 视频元数据（分辨率、帧率、总帧数）
- 每帧的差异像素数量
- 建议阈值

### .fcpxml 文件

Final Cut Pro X 标记文件，包含：
- 静态片段的起始标记（start1, start2, ...）
- 静态片段的结束标记（stop1, stop2, ...）
- 视频格式信息

导入方法：
1. 打开 Final Cut Pro X
2. 文件 → 导入 → XML...
3. 选择生成的 `.fcpxml` 文件

## 技术细节

### 性能优化

- **并发处理**：使用 goroutine 进行帧解码和处理
- **对象池**：预分配 Mat 对象，减少内存分配开销
- **进度显示**：实时显示分析进度条

### 支持的视频格式

- MP4
- MOV
- AVI
- MKV
- WMV
- FLV
- M4V
- MPG/MPEG

### 帧率支持

程序正确处理各种帧率，包括：
- 整数帧率（24, 25, 30 fps）
- NTSC 帧率（23.976, 29.97, 59.94 fps）

```bash
make uninstall  # 从系统路径卸载
make clean      # 清理构建文件
make run        # 构建并运行

```

## 注意事项

1. 首次分析视频可能需要较长时间，取决于视频长度和分辨率
2. 分析结果保存在 `.gob` 文件中，可以重复使用
3. 阈值设置过低可能误判，过高可能漏检，建议先查看统计信息
4. 最小持续时间设置可以过滤掉短暂的画面静止
